# API REST with Express framework

## Instructions

Create .env file with the environment variables

```bash
PORT=3000
JWT_KEY=5tj567trgh567456h
DB_HOST=clustername.abcdefg.mongodb.net
DB_PORT=27017
DB_NAME=dbname
DB_USER=username
DB_PASSWORD=password
```

```bash
# Install dependencies
npm i

# Insert test data
npm run seed

# Start API
npm start
```
